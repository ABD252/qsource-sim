# مسرد المصطلحات — qsource-sim
# Glossary of Terms (English ⇄ العربية)

هذا المسرد هو **المرجع المعياري** للمصطلحات في كل الوثائق العربية للمشروع.
القاعدة: عند أول ورود للمصطلح في أي نص عربي يُكتب المقابل العربي متبوعًا
بالمصطلح الإنجليزي بين قوسين، ثم يُستخدم العربي (أو الاختصار الإنجليزي
الشائع مثل RIN وJSA) في بقية النص.

## الفيزياء الأساسية

| المصطلح الإنجليزي | المقابل العربي | الشرح المبسط |
|---|---|---|
| SPDC (Spontaneous Parametric Down-Conversion) | التحويل الوسيطي التلقائي النازل | عملية في بلورة لاخطية: فوتون ضخ واحد ينشطر إلى فوتونين توأمين أقل طاقة (إشارة + عاطل) |
| photon pair | زوج الفوتونات | الفوتونان التوأمان الناتجان من انشطار فوتون الضخ |
| signal / idler | الإشارة / العاطل | الاسمان الاصطلاحيان لفوتونَي الزوج؛ نكشف أحدهما لنعلن وجود الآخر |
| heralded single photon | الفوتون المفرد المُبشَّر به | فوتون نعلم بوجوده لأن توأمه انكشف في الكاشف — "بشّرنا" به |
| heralding | التبشير | استخدام كشف أحد الفوتونين إعلانًا عن وجود الآخر |
| heralding efficiency | كفاءة التبشير | احتمال وجود الفوتون فعلًا في القناة عندما يعلن الكاشف عن توأمه |
| entanglement | التشابك | ارتباط كمي بين جسيمين لا يوصف بحالتين منفصلتين |
| pump / pump laser | الضخ / ليزر الضخ | الليزر الذي يغذي البلورة — كل عيوبه هي موضوع هذا البحث |
| nonlinear crystal | البلورة اللاخطية | الوسط الذي يحدث فيه الانشطار (عندنا: شبيهة بـ PPKTP) |
| PPKTP | بلورة KTP دورية الأقطاب | بلورة فوسفات تيتانيل البوتاسيوم بأقطاب معكوسة دوريًا لمطابقة الطور |
| collinear | مسامت | الفوتونان يخرجان في نفس اتجاه الضخ |
| degenerate | متساوي التردد | فوتونا الزوج لهما التردد نفسه (775nm → 1550+1550nm) |

## الوصف الطيفي للحالة الكمية

| المصطلح الإنجليزي | المقابل العربي | الشرح المبسط |
|---|---|---|
| JSA (Joint Spectral Amplitude) | السعة الطيفية المشتركة | الدالة f(ν_s,ν_i) التي تصف توزيع ترددَي الزوج معًا — قلب الورقة كلها |
| JSI (Joint Spectral Intensity) | الشدة الطيفية المشتركة | مربع قيمة JSA؛ ما تقيسه التجارب فعليًا |
| spectral purity (P) | النقاء الطيفي | مقياس من 0 إلى 1: هل الفوتون المُبشَّر به في حالة طيفية واحدة نظيفة؟ P=1 مثالي |
| Schmidt decomposition | تحليل شميت | تفكيك JSA إلى أنماط مستقلة؛ رياضيًا هو تحليل SVD للمصفوفة |
| Schmidt number (K) | عدد شميت | العدد الفعّال للأنماط الطيفية المختبئة في الفوتون؛ K=1/P؛ المثالي K=1 |
| SVD (Singular Value Decomposition) | تحليل القيم المفردة | التفكيك المصفوفي الذي يحسب تحليل شميت عدديًا |
| separable / factorable | قابل للفصل | JSA تُكتب حاصل ضرب دالتين مستقلتين — شرط النقاء الكامل |
| spectral (anti-)correlation | الترابط الطيفي (العكسي) | معرفة تردد أحد الفوتونين تكشف تردد الآخر — عدو النقاء |
| detuning (ν) | الانحراف الترددي | بُعد التردد عن مركز التصميم؛ محاور كل خرائطنا |
| energy conservation | حفظ الطاقة | ترددا الزوج يجمعان لتردد الضخ — يرسم الشريط القطري في JSA |
| momentum conservation / phase matching | حفظ الزخم / مطابقة الطور | شرط البلورة — يرسم حافة sinc المائلة في JSA |
| phase-matching function φ | دالة مطابقة الطور | شكل sinc(ΔkL/2) الناتج من طول البلورة المحدود |
| sinc function | دالة sinc | sin(x)/x — شكلها ذو فصوص جانبية تحد النقاء المثالي بـ 0.81 |
| GVM (Group-Velocity Mismatch) | عدم تطابق السرعة الزمرية | فرق سرعات انتشار الضخ والفوتونين داخل البلورة (κ_s وκ_i) — يحدد ميل الحافة |
| apodization | التلطيف (apodization) | هندسة البلورة لتحويل sinc إلى غاوسي وإزالة الفصوص — يرفع السقف فوق 0.81 |
| Sellmeier equations | معادلات سلمايِر | صيغ دقيقة لمعامل الانكسار حسب الطول الموجي (خارج نموذجنا الحالي) |
| chirp | الزقزقة (chirp) | تغير التردد داخل النبضة الواحدة؛ طور طيفي غير مسطح (عمل مستقبلي) |
| transform-limited | محدود التحويل | نبضة مثالية: أقصر مدة ممكنة لعرض طيفها؛ طور مسطح |

## الحالات المختلطة ومونتي كارلو

| المصطلح الإنجليزي | المقابل العربي | الشرح المبسط |
|---|---|---|
| pure state | الحالة النقية | حالة كمية موصوفة بدالة موجة واحدة |
| mixed state | الحالة المختلطة | خليط إحصائي كلاسيكي من حالات نقية — هذا ما ينتجه الليزر المعيب |
| density matrix (ρ) | مصفوفة الكثافة | الوصف الرياضي للحالة المختلطة: ρ = Σ wₖ AₖAₖ† |
| ensemble | الحشد الإحصائي | مجموعة التحقيقات التي يخلطها الليزر المعيب |
| Monte-Carlo (MC) | مونتي كارلو | حساب بالعينات العشوائية: نولّد K تحقيقًا للضخ ونجمع مصفوفة الكثافة |
| realization | التحقيق | عينة واحدة من الحشد: ضخة واحدة بمركز وإزاحة وقدرة معينة |
| seed | البذرة | رقم يثبّت العشوائية حتى تكون النتائج قابلة لإعادة الإنتاج حرفيًا |
| SEM (Standard Error of the Mean) | الخطأ المعياري للمتوسط | شريط الخطأ الأمين: تشتت البذور ÷ √عددها |
| uncertainty | الارتياب | شريط الخطأ المصرَّح به مع كل ميزانية؛ في هذا البحث هو المتوسط ± SEM |
| confidence interval | فترة الثقة | المدى الذي نثق أن القيمة الحقيقية داخله |
| quasi-static | شبه ساكن | افتراضنا الزمني: الضوضاء أبطأ من النبضة وأسرع من زمن القياس |
| coherence time | زمن التماسك | المدة التي يبقى فيها الضوء "متذكرًا" طوره (~1 ps لضخنا) |
| closed form | الصيغة المغلقة | حل رياضي صريح بلا حاسوب — سلاح التحقق الخارجي عندنا |
| Gauss–Hermite quadrature | تربيع غاوس–هرميت | تكامل عددي حتمي عالي الدقة استُخدم للتحقق من الصيغة |
| finite-K bias | انحياز K المحدود | خطأ منهجي صغير (∝1/K) في مقدّر مونتي كارلو؛ حسبناه وضممناه |

## عيوب الليزر (موضوع الورقة)

| المصطلح الإنجليزي | المقابل العربي | الشرح المبسط |
|---|---|---|
| imperfection budget | ميزانية العيوب | الجدول المركزي: أقصى عيب مسموح قبل خسارة 1%/5%/10% من النقاء |
| tolerance | السماحية / التحمل | كم يتحمل المصدر من العيب قبل تجاوز الميزانية |
| datasheet | ورقة البيانات | نشرة المواصفات التي ينشرها مصنّع الليزر مجانًا — مصدر أرقامنا |
| center-frequency jitter | رجرجة التردد المركزي | اهتزاز مركز الضخ ضخةً بعد ضخة (rms بالـ GHz) — المواصفة الأصعب |
| drift | الانجراف | زحف بطيء للمركز (حرارة، تيار) — يعامَل مثل الرجرجة في نموذجنا |
| RIN (Relative Intensity Noise) | ضجيج الشدة النسبي | تذبذب قدرة الليزر؛ لا يمس النقاء إطلاقًا لكنه يفسد إحصاء الأزواج |
| (exactly) invariant | ثابت (تمامًا) | لا يتغير البتة؛ الصيغة المعتمدة لنتيجة RIN المركزية: «يترك النقاء ثابتًا تمامًا» |
| order(s) of magnitude | مرتبة عشرية / مراتب عشرية | عامل ×10؛ «أدنى بخمس مراتب عشرية» = أدنى بعامل 10⁵ |
| linewidth | عرض الخط | العرض الطيفي لليزر cw؛ اللورنتزي منه ضجيج طور سريع لا يدخل نموذجنا إلا كحد أعلى |
| Lorentzian / Gaussian lineshape | شكل الخط اللورنتزي / الغاوسي | شكلا الخط الشائعان؛ التمييز بينهما جوهري عند الترجمة |
| FWHM | العرض الكامل عند منتصف القمة | طريقة المصنّعين في قياس العرض؛ نحوّله إلى σ عبر ÷1.665 |
| rms (root mean square) | جذر متوسط المربعات | مقياس التذبذب الإحصائي المستخدم في كل الميزانيات |
| longitudinal modes | الأنماط الطولية | خطوط التجويف المتعددة — ليزر "متعدد الأنماط" يبث مشطًا لا خطًا |
| mode spacing / FSR | تباعد الأنماط / المدى الطيفي الحر | المسافة بين خطوط المشط = c/2nL للتجويف الخطي |
| frequency comb | المشط الترددي | طيف من خطوط متساوية التباعد |
| mode-locked laser | ليزر مقفول الأنماط | أنماطه متماسكة الطور → نبضات؛ مشطه حالة نقية لا خليط (خارج نموذج المشط عندنا) |
| SMSR (Side-Mode Suppression Ratio) | نسبة كبح النمط الجانبي | كم ديسيبل أخفض النمط الجانبي من الرئيسي — تتحول عندنا إلى وزن الخط |
| FP (Fabry–Pérot) diode | دايود فابري–بيرو | أرخص الليزرات: حر الجريان متعدد الأنماط — يفشل في ميزانيتنا بالبناء |
| DFB (Distributed FeedBack) | ليزر التغذية الراجعة الموزعة | ليزر أحادي التردد مستقر — ممتاز للاستقرار، خاطئ للنقاء (cw ضيق جدًا) |
| ECDL (External-Cavity Diode Laser) | دايود خارجي التجويف | ليزر مضبوط بعرض خط ضيق جدًا |
| TBP (Time-Bandwidth Product) | حاصل المدة×العرض | ثابت النبضة المثالية (0.315 لنبضات sech²) |
| pulse energy noise | ضجيج طاقة النبضة | صورة RIN في الليزرات النبضية |

## القياس والتداخل

| المصطلح الإنجليزي | المقابل العربي | الشرح المبسط |
|---|---|---|
| HOM (Hong–Ou–Mandel) interference | تداخل هونغ–أو–ماندل | فوتونان متطابقان يدخلان مقسّم حزمة يخرجان معًا دائمًا — مقياس التطابق الذهبي |
| HOM dip | غطسة HOM | انخفاض التطابقات عند تراكب الفوتونين — عمقها = الرؤية |
| visibility (V) | الرؤية | عمق التداخل من 0 إلى 1؛ لمصدرين مستقلين متطابقين V = P |
| two-source HOM | تداخل HOM بين مصدرين | التجربة التي تقيس فعليًا الكمية التي تحدّها ميزانياتنا |
| NOON state / de Broglie wave | حالة NOON / موجة دي برولي الفوتونية | تداخل ثنائي الفوتون بدور نصف الطول الموجي — حيث تظهر بصمة مشط الضخ |
| CAR (Coincidence-to-Accidental Ratio) | نسبة التطابقات إلى العرضيات | جودة مصدر الأزواج إحصائيًا؛ RIN يخفضها عبر معزِّز الأزواج المضاعفة |
| accidental coincidences | التطابقات العرضية | تطابقات زائفة من زوجين مختلفين — تزداد مع تذبذب القدرة |
| double-pair emission / boost | انبعاث/معزِّز الأزواج المضاعفة | إنتاج زوجين في نفس النبضة؛ RIN يضخمه بعامل ⟨w²⟩/⟨w⟩² ≈ 1+r² |
| pair rate (μ) | معدل الأزواج | متوسط عدد الأزواج في النبضة |
| g²(0) | دالة الترابط الثنائي | قياس إحصائي للنقاء حساس للطور |
| JSI characterization | توصيف الشدة الطيفية المشتركة | قياس خريطة JSI معمليًا (مطيافات، ألياف مشتتة...) |
